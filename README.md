# Student Attendance Analysis & Prediction System

## Overview
This project predicts whether a student is Regular or Irregular based on study habits and attendance-related features.

## Features
- Student Prediction
- Interactive Streamlit Dashboard
- Attendance Analysis
- Data Visualization
- Machine Learning Model

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib
